# ShopSmart Analytics Pipeline 📊

## 🚀 Overview
Built an end-to-end ETL pipeline to simulate how e-commerce data is processed and analyzed for business decision-making.

## ⚙️ What This Project Does
- Extracts live data from 3 REST APIs using Python  
- Validates incoming data using Pydantic  
- Cleans and transforms data using Pandas  
- Stores structured data in a SQLite database  
- Analyzes data using SQL (JOINs, CTEs, window functions)  
- Visualizes insights using Matplotlib  

## 📊 Key Insights
- Identified top consumers contributing the highest revenue  
- Analyzed customer and product performance trends  
- Highlighted opportunities for improving business strategy  

## 🛠 Tech Stack
- Python (Pandas, Requests, Matplotlib)  
- SQL  
- SQLite  
- Pydantic  

## 📁 Project Structure
- `Git hub Project 1.ipynb` → Full analysis and pipeline  

## 💡 Business Impact
This project demonstrates how raw data can be transformed into actionable insights to support data-driven decision-making in an e-commerce environment.
